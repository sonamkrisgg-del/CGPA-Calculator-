# cgpa_calculator

## Overview
This project is a **CGPA (Cumulative Grade Point Average) Calculator** written in C++. It allows students to calculate their CGPA based on grades and credit hours for different subjects. The program demonstrates core C++ concepts such as loops, arrays, and functions.

## Features
- Input grades and credit hours for multiple subjects
- Calculate GPA and CGPA
- User-friendly text-based interface
- Accurate weighted average calculation

## Requirements
- C++ compiler (e.g., GCC, Clang, or Visual Studio)
- Basic understanding of C++ programming

## How to Compile and Run
1. Open a terminal or command prompt.
2. Navigate to the folder containing the source file.
3. Compile the program with the following command:
   ```bash
   g++ "cgpa_calculator.cpp" -o CGPACalculator
   ```
4. Run the compiled program:
   ```bash
   ./CGPACalculator
   ```

## Example Usage
```
Enter number of subjects: 5
Enter grade points and credit hours for each subject...
Your CGPA is: 8.7
```

- `cgpa_calculator.cpp`: Main source file containing the CGPA calculator logic.
